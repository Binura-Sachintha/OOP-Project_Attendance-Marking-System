**Changes from V2-**

-  Data will be now stored in a SQL Server instead of .ser files

### **_Before you run the main.java file make sure to follow below steps-_**

1. Open Sql Server Configuration Manager
2. Double click on Sql Server Network Configuration
3. Double click on Protocols for MSSQLSERVER
4. Right click on TCP/IP and Enable it
5. Then double click on Sql Server Services
6. Right click on SQL SERVER(MSSQLSERVER) 
7. Press Restart


After above steps-

- **_Open the Sql Server Management Studio and run the sql query attached_**

[RUN THIS IN SQL SERVER MANAGEMENT STUDIO.sql](https://github.com/user-attachments/files/23848320/RUN.THIS.IN.SQL.SERVER.MANAGEMENT.STUDIO.sql)
